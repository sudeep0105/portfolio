03 — Project

Featured Work

Machine Learning Healthcare Tech

Classification of Sleep Patterns & Disorders

The Challenge

Traditional sleep diagnostic methods like Polysomnography (PSG) are often inconvenient and expensive for patients. This project explores using machine learning to classify sleep disorders directly from health and lifestyle data sourced from Kaggle — making diagnosis more accessible.

Technical Approach

Implemented Ensemble Learning techniques — specifically Stacking and Voting Classifiers — to enhance prediction accuracy and model robustness. This outperformed single-model baselines like KNN and SVM by leveraging the strengths of multiple learners.
